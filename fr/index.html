Qu'est-ce que le développement front-end?

Le développement front-end vous permet de créer de beaux sites bien codés.

Que vais-je apprendre à faire?

Vous apprendrez comment rapidement transformer des idées en prototypes, ainsi que construire des sites web bien structurés.

Quels sont les requis?

Ce cours est pour débutants désireux d'apprendre le développement web. Il vous sera demandé de mettre beaucoup d'effort personnel afin de profiter pleinement de la formation.
Un ordinateur portable est également requis.

Curriculum

  • Aperçu de l'horaire
  • Mise en place de l'environnement
  • Aperçu des rôles du HTML et CSS
  • Aperçu de Git et GitHub
  • Bref survol des technologies web
  • Bref historique du langage HTML
  • Introduction au rôle du HTML, structure vs. présentation
  • Discussion à propos de HTML5
  • Utiliser les balises HTML sémantiques afin de créer du contenu annoté
  • Discussion sur les différents formats d'images
  • Introduction au rôle du CSS, structure vs. présentation
  • Expérimentation avec les propriétés de base du CSS
  • Compréhension de la cascade et spécificité du CSS
  • Compréhension du Box Model (modèle de boîtes) du CSS
  • Utiliser l'inspecteur du navigateur
  • Utiliser les Media Queries (requêtes média) pour des sites réactifs
  • Expliquer l'utilisation du Javascript dans le développement front-end
  • Discussion sur la programmation déclarative vs. impérative
  • Le langage Javascript: variables, données, contrôle, fonctions
  • Introduction au DOM (Modèle Objet du Document) et à sa manipulation
  • Utiliser jQuery pour faciliter la manipulation du DOM
  • Discussion sur le web réactif
  • Exploration de Foundation basée sur les connaissances acquises à présent
  • Démo: créer une petite page d'accueil avec les composantes Foundation

En utilisant toutes les connaissances acquises durant la formation, les participants créeront un projet de leur choix. Ce projet peut être exécuté seul ou en groupe. Votre instructeur contrôlera l'étendue du projet et vous guidera durant toute la semaine.
