Welcome to DecodeMTL :)

This preparatory material is the first step in your journey to become a web developer. The material is designed to get your feet wet with web development, and make sure our intensive bootcamp is a good fit for you.

Objectives

By going through this preparatory material, you should be fully up to speed to start the bootcamp. You will learn the following things:

  • The very fundamentals of HTML and CSS
  • Some cool tricks to make you a CSS-ninja
  • The very fundamentals of JavaScript
  • How to use Git and GitHub to manage code and collaborate with a team
  • The very fundamentals of your computer's command line

Commitment

This preparatory material will take anywhere between 30-40 hours to complete. Completing this material before starting the bootcamp is mandatory. Failing to do so, you will be playing a catch-up game during the actual bootcamp.

That being said, you are not expected to complete all this material by yourself. If you get stuck, be sure to get in touch with us for hints and/or explanations.
