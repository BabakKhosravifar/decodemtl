What is front-end development?

Front-end web development allows you to build beautifully crafted and well coded simple websites.

What will I learn how to do?

You will learn how to quickly turn ideas into working prototypes, and be able to design and build a well structured website.

What are the course requirements?

This course is for beginners who are eager to learn web development. You will be required to put in a lot of work to get the most out of this course.
A laptop is also required.

Course
Curriculum

  • Overview of the schedule
  • Setup of development environment
  • High-level introduction to HTML and CSS
  • High-level introduction to Git and Github
  • Brief overview of how the web works (servers, clients, etc.)
  • Brief history of HTML
  • Introduction to HTML's role, structure vs. presentation
  • Understanding what is meant by HTML5
  • Using HTML semantic tags to create meaningful documents
  • Understanding of images, differences between the various image formats
  • Introduction to CSS' role, structure vs. presentation
  • Experimenting with basic CSS properties
  • Understanding of the CSS cascade and specificity
  • Understanding of the CSS box model
  • Use a browser's inspector to interact with the content
  • Use media queries to create a responsive experience
  • Explain the use of Javascript in front-end development
  • High-level understanding of imperative vs. declarative coding
  • Javascript language: variables, data types, control flow, functions
  • Introduction to the DOM and its manipulation
  • Using jQuery for smoother DOM manipulations

Using everything that was learned through the sessions, create a personal landing page or a microsite. This project can be done alone or with a group. Your instructor will control the scope of the project, and will be guiding you throughout.
